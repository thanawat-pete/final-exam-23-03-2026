# วิธีการใช้งาน (User Manual)

คู่มือนี้จะอธิบายฟีเจอร์ต่างๆ และวิธีการใช้งานระบบ **SE NPRU TaskFlow** ในรูปแบบ **Shared Board (ทุกคนเห็นงานร่วมกัน)**

---

## 1. การสมัครสมาชิกและเข้าสู่ระบบ (Authentication)

### 1.1 สมัครสมาชิก (Sign Up)
- ไปที่หน้า **Sign Up**
- กรอกชื่อนามสกุล, อีเมล (Email), และรหัสผ่าน (Password)
- กดปุ่ม **Sign Up** เพื่อลงทะเบียน บัญชีของคุณจะถูกเก็บใน MongoDB Atlas

### 1.2 เข้าสู่ระบบ (Log In)
- กรอกอีเมลและรหัสผ่านที่สมัครไว้
- หากคุณใช้บนระบบ Production (Render/Vercel) ตัว Cookie จะถูกตั้งค่าเป็น `Secure` และ `SameSite: None` เพื่อความปลอดภัย

---

## 2. การจัดการงาน (Task Management)

### 2.1 สร้างงานใหม่ (Create Task)
- กดปุ่ม **+ Create New Task**
- กรอกชื่อเรื่อง (Title), คำอธิบาย (Description), และเลือกวันครบกำหนด (Due Date)
- กด **Save** งานจะปรากฏในรายการ

### 2.2 จัดการสถานะงาน (Task Status)
- คุณสามารถเปลี่ยนสถานะงานได้ 3 รูปแบบ:
  - **Pending (รอดำเนินการ)**: งานเริ่มต้น
  - **In Progress (กำลังทำ)**: งานที่กำลังดำเนินการอยู่
  - **Completed (เสร็จสิ้น)**: งานที่ทำเสร็จแล้ว

### 2.3 การแก้ไขและลบ (Edit & Delete)
- สามารถกดที่รูป **ถังขยะ** เพื่อลบงาน
- หรือคลิกเข้าไปในงานเพื่อแก้ไขรายละเอียด

---

## 3. ฟีเจอร์เพิ่มเติม (Extra Features)

### 3.1 การเปลี่ยนภาษา (Multi-Language)
- สามารถสลับระหว่างภาษา **ไทย (TH)** และ **อังกฤษ (EN)** ได้ที่ไอคอนเปลี่ยนภาษาด้านบน

### 3.2 การเปลี่ยนธีม (Dark Mode / Light Mode)
- เปลี่ยนโหมดแสดงผลเป็นโหมดมืด (Dark) หรือโหมดสว่าง (Light) ตามต้องการ

### 3.3 แถบความคืบหน้า (Progress Bars)
- ด้านบนแสดงการสรุปงานทั้งหมดแยกตามสถานะ เพื่อให้คุณเห็นภาพรวมของงานที่เหลืออยู่

---

## 4. ข้อควรระวังและการแก้ไขเบื้องต้น

- **ปัญหา Login ไม่ได้**: ตรวจสอบว่า `NODE_ENV` ใน Server ถูกตั้งเป็น `production` หรือไม่ (ถ้าเป็นบน Render) และตรวจสอบว่าเปิดใช้ `Secure Cookie` บนเครื่องคอมพิวเตอร์ของคุณแล้วหรือไม่
- **การเชื่อมต่อ**: หากหน้าเว็บโหลดงานไม่ขึ้น ให้เช็คว่า Backend บน Render รันอยู่หรือไม่ (Render ตัวฟรีอาจจะใช้เวลาตื่นหรือ Resume ประมาณ 30-50 วินาที)
